Conversor de Monedas API
El proyecto es una API construida con que permite convertir distintas monedas utilizando tasas de cambio actualizadas.
///Tecnologías///
-Java 17
-Maven
-IntelliJ IDEA Community Edition

///Funcionalidades///
Convierte un monto de una moneda a otra usando tasas de cambio actualizadas.
Parámetros: moneda origen, moneda destino, cantidad a convertir.
Devuelve: monto convertido, moneda origen y destino, y la tasa utilizada.
